#### Ночь, улица, фонарь, аптека… 

Ночь, улица, фонарь, аптека,   
Бессмысленный и тусклый свет.  
Живи еще хоть четверть века —  
Всё будет так. Исхода нет.

Умрёшь — начнёшь опять сначала  
И повторится всё, как встарь:  
Ночь, ледяная рябь канала,  
Аптека, улица, фонарь.  

_Александр Блок_
